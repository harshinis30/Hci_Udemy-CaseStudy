# 🎓 Udemy UI/UX Redesign — HCI Case Study

## 📌 Overview

This project presents a **complete Human–Computer Interaction (HCI) analysis and redesign** of the Udemy website.
The goal was to identify usability issues in the existing interface and improve the user experience using established HCI principles.

The redesign was created as an **interactive Figma prototype** supported by a detailed evaluation report.

---

## 🎯 Objectives

* Analyze the existing Udemy interface using HCI frameworks
* Identify usability issues across key user flows
* Redesign screens to improve:

  * Usability
  * Accessibility
  * Efficiency
  * User satisfaction
* Map every design decision to **recognized HCI principles**

---

## 🧠 HCI Principles Applied

This project is grounded in:

* **Nielsen’s Usability Heuristics**
* **Shneiderman’s Eight Golden Rules**
* **Norman’s Design Principles**
* **Gestalt Principles**
* **Cognitive Psychology Concepts**

---

## 🖥️ Screens Redesigned

The redesign covers the full user journey:

1. Landing Page
2. Login Page
3. Registration Flow
4. Learner Dashboard
5. Home Feed
6. Video Player
7. Category Navigation
8. Profile & Settings
9. Help & Support
10. Empty Search State
11. Search Results
12. Add to Cart
13. Shopping Cart
14. Checkout Flow

---

## 🔍 Key Improvements

### ✨ Usability Enhancements

* Reduced cognitive load with **step-by-step flows**
* Improved navigation using **clear hierarchy & grouping**
* Added **real-time validation and feedback**

### ⚡ Efficiency Improvements

* Faster course discovery with **filters & suggestions**
* Resume learning features
* Keyboard shortcuts in video player

### 🚫 Error Prevention & Recovery

* Input validation before submission
* Smart suggestions for failed searches
* Multiple recovery paths (retry, browse, support)

### 🎯 User Experience

* Minimalist and clean interface
* Better visual hierarchy
* Clear CTAs and feedback mechanisms
